# Patient Visits Analytics Dashboard
Healthcare analytics dashboard built using SQL Server and Power BI to analyze patient utilization and diagnoses.

## Tech Stack
- SQL Server
- Power BI
- Excel (source data)

## Key Insights
- 200 visits across 42 patients
- Average of 4.76 visits per patient
- Chronic disease management drives recurring visits.
- A core patient group contributes heavily to overall visit volume.
- High-utilization patients may benefit from proactive monitoring and structured care strategies.
  
## Dashboard Highlights

### Age Distribution

### Top Diagnoses
- **Hypertension**
- **Asthma**
- **Hyperlipidemia**
- **Diabetes**
- Chronic conditions account for the highest visit volumes, suggesting ongoing care management rather than isolated acute events.

### High Utilizers
- Patients with **4 or more visits** represent a small subset of the population but account for a disproportionate share of total visits.
- This pattern highlights potential opportunities for care coordination and targeted intervention strategies.
